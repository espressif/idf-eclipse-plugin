ESP-IDF Eclipse Plugin

If you are installing IDF Eclipse Plugin into your Eclipse for the first time, you need to add the new release's repository as follows:

  • Help > Install New Software... 
  • Click Add...
  • Enter the URL of the new repository 
    • stable: https://dl.espressif.com/dl/idf-eclipse-plugin/updates/latest/
    • beta: https://dl.espressif.com/dl/idf-eclipse-plugin/updates/beta/
    • nightly: https://dl.espressif.com/dl/idf-eclipse-plugin/updates/nightly/
  • Click Ok

If you've already installed IDF Eclipse Plugin using update site URL, you can get the latest changes using below

  • Help > Check for Updates
  • If updates are found, select Espressif IDF Plugins for Eclipse and deselect all other items
  • Click Next to proceed with the installation

The IDF Eclipse Plugin is available from the Eclipse Marketplace. The Marketplace client can be used to search for and install the ESP-IDF Eclipse Plugin.
